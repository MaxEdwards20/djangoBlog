HTML Elements

  1. One of the elements is <li> . This is used as a list item; so every item in a list must be related to one of these list items. This is a block level element.
  2. Another element is the <ul>. This is short for unordered list. Basically just bullet points, this helps organize information. This is a block level element.
  3. The paragraph element is shown as <p>. It's purpose is to format text for any sized window so that any device or screen setup will still alow users to see everything without words falling off of the page. This is a block level element.
  4. The header one element is shown as <h1>. This creates a large text size that helps users see different subjections thorughout a page. This is a block level element.

HTML Character Entities

  1. The equal sign is incredibly helpful for math and general logic. It looks like this =
  2. The fraction symbol can be used to display fractions. For example, you can do one fourth ¼, or three fifths, ⅗, and other combinations. This can even be done within other blocks as shown here (inside a code block) ⅗ + ⅖ = 1;
  3. The less than symbol is used to show that something is worth less than another object, usually a number. It is also a character used to show the literal tags used when writing html. It looks like this: <

CSS Selectors

The four CSS selectors used on this site are as follows:
  1. Element This is used in a global style because it affects every element of that type; for example I have a few element selectors to get paragraphs, header one, and the body of every page.
  2. .class This is used to change specific, repeated items. I created a blogPost class that formats everything with the same background colors and padding to look uniform. When something is named or tagged with this class's ID, it gets whatever styling information is stored within this class on the css sheet. All of the monospaced elements you see are matched to the id mono which allows them to look uniform. This is more specific than the class indicator.
  3. #id Id must be a unique element across the entire project. In the Main Blog page there is a special id Carbs that looks different than the rest of the words around it; this is because of a unique id.
  4. :nth-child This is used across each page to switch the background color of every other list item. It can be adjusted to start on the even or odd child, depending on what you want.

CSS Properties

  • background-color is used to change the background color from blue, to black, to orange, to whatever other color you would like to see. Two possible forms are just a color, like blue, or a hexadecimal value, like #FFD700.
  • font-size is used to change the size of the font. It can be changed with the value smaller or with a specific size like 12em.
  • color is used to change the color of the actual words. Rather than a highlighter like background color, this changes words as they are used. Notice how the headers are all orange while the body text is black and white; this is because of a color value being changed. Like background-color, this can be set to a nomral color like red, or a hexadecimal value like #FFD700
